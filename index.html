PIZZA BILLING SYSTEM

In our third semester, we were assigned the task of creating a Java-based pizza billing software integrated with a database. The objective was to develop a system that could take orders, calculate bills, and store transaction details in a database for record-keeping. I designed a user-friendly interface with features for selecting pizza options, customizing toppings, and generating invoices. Using Java for the front-end and MySQL for the database, I successfully implemented a functional and efficient system, demonstrating my ability to connect programming logic with data management.
